Method

Text-as-data layer

The app tokenizes documents, computes dictionary scores for market-relevant themes, ranks evidence passages, and compares source groups. It can run locally.

Private relay

The app calls a private backend adapter when available. Provider keys stay in environment variables, and the interface never asks the user for secrets.

Research boundary

Outputs are evidence summaries for research. The app does not forecast prices, recommend trades, or automate portfolio decisions.
